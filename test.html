WordPress Math Query Database Library

This library handles mathematical queries on a database table in WordPress. You can perform a wide range of calculations and aggregations using the Math_Query class.

Installation and Setup

To use this library, include it in your WordPress project using Composer. Add the following to your composer.json file:

{
  "require": {
    "arraypress/math-query": "*"
  },
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/arraypress/math-query"
    }
  ]
}

After adding the dependency, run composer install --no-dev to install the library.

Using the Plugin Meta Library

You can use the Math_Query class to perform mathematical queries on your WordPress database table. Here's how you can use it:


require_once dirname(__FILE__) . '/vendor/autoload.php';

    
$query = array(
    'table'      => 'your_table_name',
    'column'     => 'amount',
    'function'   => 'SUM',
    'group_by'   => 'status',
    'date_start' => '2023-01-01',
    'date_end'   => '2023-12-31',
    'context'    => 'monthly_sales'
);

    
$math_query = new \ArrayPress\Utils\Math_Query($query);

    
$result = $math_query->get_result();

Query Parameters

Here's a breakdown of the query parameters:

Parameter Description Example
table The name of the database table you want to query. 'your_table_name'
column The name of the column you want to perform calculations on. 'amount'
function The mathematical function to apply (e.g., 'SUM', 'AVG', 'MAX'). 'SUM'
date_start The start date for filtering the data. '2023-01-01'
date_end The end date for filtering the data. '2023-12-31'
group_by (Optional) Group the results by a column (e.g., 'status'). 'status'
context (Optional) A custom context to identify the query. 'monthly_sales'

You can modify the query parameters to perform various calculations and aggregations on your database table.
